# Graphic Design Portfolio Website (React SPA)

This portfolio is a single-page application (SPA) built with **Vite + React + React Router**.

## Features

- **Responsive design**
- **Smooth animations** (reveal on scroll, count-up metrics, subtle parallax)
- **Projects modal**
- **Client-side routing**
  - `/` Home
  - `/projects` Archive
  - `/blogs` Writing

## Tech stack

- **React**
- **React Router**
- **Vite**
- **CSS** (existing `style.css` and `mediaqueries.css`)

## Demo

## Local development

```bash
npm install
npm run dev
```

Then open the URL printed by Vite (usually `http://localhost:5173/`).

## Production build

```bash
npm run build
npm run preview
```
